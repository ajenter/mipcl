Using MIPCL with MIPLIB2010 test engine

The MIPLIB2010 is a test set of challenging MIP problems. MIPLIB2010 is used as a test suite for mixed-integer programming algorithms. It is also used for comparison MIP solvers.

Since all problems in MIPLIB2010 archives are written in the MPS format, we wrote a programs, named mps_mipcl, that reads MIP tests in that format and then solves those tests using the multithreaded MIPCL library.

  • To get MIPLIB2010 test engine running with mps_mipcl, follow the instructions given in $MIPLIB2010/README.
  • To fulfill Step 2 of those instruction, run the following commands:
    cd $MIPLIB2010
    ln -s $MIPCLDIR/bin/mps_mipcl ./bin/mipcl<br>
    cp ./miplib2010_scripts/* ./scripts<br>
    * make SOLVER=mipcl TIME=7200 test
    *

Here it is assumed that $MIPLIB2010 points to the root directory of the MIPLIB2010 test engine, and $MIPCLDIR points to the installation directory of MIPCL.
