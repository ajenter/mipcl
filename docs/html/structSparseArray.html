Detailed Description

This structure has been designed for storing sparse arrays.

This structure implements a simple but very efficient structure for storing sparse arrays. It has been designed just for using within MIPCL. For efficiency reasons, SparseArray lacks a lot what is needed for safely using in external applications.

It is up to the user to allocate memory for three internal plain arrays.
