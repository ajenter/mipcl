Mixed Integer Programming Class Library (MIPCL)

What is MIPCL

MIPCL is a comprehensive tool designed to help you build and solve mixed-integer programming models quickly, easily, and efficiently. MIPCL is currently the fastest non-commercial mixed integer programming solver. It can be used together with a modeling tool named MIPshell. MIPCL allows the user to totally control the solution process. The library can be used for implementing algorithms of solving complex real world applications.

Both, single-threaded and multi-threaded, MIPCL libraries are available.

Features

MIPCL supports preprocessing, symmetry breaking, primal heuristics, automatic cut/column generation, and strong branching.

The MIPCL API makes it easy for you to integrate optimization into your own application. The developer interface has been designed for maximum ease-of-use. MIPCL is highly flexible through many virtual functions that allow developers to implement problem specific

  • separation routines to apply problem specific cutting planes;
  • column generation procedures;
  • primal heuristics to search for feasible solutions with specific support for probing and diving;
  • node selectors to guide the search;
  • branching rules to split the problem into subproblems;
  • preprocessing to simplify solved problems;
  • pool management of cuts and generated columns.

MIPCL comes with clear, comprehensive documentation and examples to help you get started quickly.

MIPCL supports implementation of both branch-and-cut and branch-and-price algorithms.

MIPCL is implemented as a library of C++ classes, only a few of them are public. The main classes are CMIP and CLP. CMIP is a system for solving Mixed Integer Programs (MIPs), and CLP is a system for solving Linear Programs (LPs).

MIPCL can be used

  • standalone as a shared object (dynamically linked library) in C++ projects;
  • together with a modeling environment MIPshell;
  • as a Python module called MIPCL-PY;
  • as a Java package called MIPCL-JAVA;
  • as an application for solving MIPs written in the MPS format.

Platforms

MIPCL is completely implemented in C++. The code should compile with any ANSI compliant C++ compiler. We have tested MIPCL with compilers from

  • GNU
  • Intel
  • Microsoft

MIPCL works on computers with 32-bit as well as 64-bit processors.

MIPCL works in parallel on multi-core computers (see Multithreading).
